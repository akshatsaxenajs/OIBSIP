# Temperature Converter

## 📌 Description
A temperature converter that converts values between Celsius, Fahrenheit, and Kelvin instantly using JavaScript.

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript

## ✨ Features
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Kelvin conversions
- Instant conversion
- Responsive design
